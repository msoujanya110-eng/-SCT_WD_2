# -SCT_WD_2
Task Description

The objective of this task was to create a Stopwatch Web Application using HTML, CSS, and JavaScript.
The stopwatch should allow users to start, pause, reset the timer, and track lap times.

Project Description

In this project, I developed a functional stopwatch web application that allows users to measure time accurately. The application includes features such as starting the timer, pausing it, resetting the timer, and recording lap times.

The user interface was designed using HTML and CSS, while the stopwatch functionality was implemented using JavaScript. The application provides a smooth and interactive experience for users.

Features

Start the stopwatch
Pause the stopwatch
Reset the timer
Record lap times
Display time in hours, minutes, seconds, and milliseconds
Simple and user-friendly interface

Technologies Used

HTML5 – Used to structure the web page
CSS3 – Used for styling and layout
JavaScript – Used to implement stopwatch functionality and timing logic
